11. Lookup Functions
VLOOKUP, HLOOKUP (legacy)
INDEX + MATCH (industry-preferred)
XLOOKUP (modern & powerful)
LOOKUP

VLOOKUP : 
=VLOOKUP("value jiska search krna hai","pura table select jismai search krna hai",konse column ka value chahiye “,"range lookup”

HLOOKUP : Similar to VLOOKUP only for horizontal rows

Index + Match : 
Syntax : =INDEX(kis range se , MATCH(“naam kiska chahiye”, (“jis range se naam milega”))

LOOKUP : Syntax:
=LOOKUP(“kiske close ek number uske close wala value”,”phle value konse range ka hai”,”konse range se ans print krna hai”)

Sort the table and then use LOOKUP

Best to use INDEX + MATCH
